Après avoir décidé de votre identité graphique, il est temps de créer votre présence en ligne. La conception d’un site internet est cruciale dans un monde où tout le monde est connecté.

Avec les technologies ci-dessous, je vous propose un site sur mesure.

Aperçu de mes derniers projets

Explication : HTML

L’HTML, ou Hypertext Markup Language, pour faire court, est simplement le langage qui permet de créer des documents que nous posterons en ligne.

Il permet d’écrire, par exemple, ce texte. Il crée la structure de votre site en créant des éléments soit en bloc, soit en ligne en fonction du résultat souhaité.

Explication : CSS

Le CSS, pour “Cascading Style Sheet” est, pour vulgariser, le langage qui permet de venir rajouter l’esthétique à votre site.

Il me permet de faire ce cadre, par exemple. C’est cette technologie qui va nous permettre aussi de choisir la typographie de votre site, et d’agencer les éléments comme bon nous semble.

Explication : JavaScript

Le JavaScript permet de rajouter un peu tout ce que l’on peut imaginer. Des animations 2D/3D tout comme des fonctions pour par exemple, créer des barres de recherche.

Il est un langage essentiel pour la gestion de commerce électronique ou de réseau social. Sans ce langage, les applications mobiles n’existeraient pas.

Explication : Bootstrap

Bootstrap est une technologie qui simplifie grandement le travail en CSS, permettant de gagner en rapidité et en efficacité tout en conservant un haut niveau de qualité.
Il existe d’ailleurs plusieurs outils qui suivent ce même principe, mais Bootstrap reste l’un des plus connus, puisqu’il a été créé par les fondateurs de Twitter
(Qui aujourd'hui à été racheté et éxiste désormais sous le nom de "X").
